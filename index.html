Portal FEUAN

Portal FEUN é um projecto web desenvolvido usando a liguagem PHP e banco de dados Mysql. É um portal universitário totalmente dinâmico composta pela parte do Front-Office e Back-Office. Na parte do Back-Office podemos encontrar as opções de Gestão e cadastro de Usuários, Cursos e Notícias. Na parte do Front-Office os estudantes podem aceder e visualizar seus perfís acadêmicos e consultar calendário acadêmico, pautas, programa curricular e horários dentre outras opções.

Forum NJ

Forum NJ é um projecto web desenvolvido usando a liguagem PHP e banco de dados Mysql. O Forum NJ permite a partilha de idéias entre usuários. No forum os usuários tem a opção de criar contas, fazer login, editar seu perfil, visualizar usuários, fazer posts ou comentários e respectivas edições e remoção dos dados. Na página inicial podemos visualizar automaticamento no feed de notícias todos os posts e comentários feitos na plataforma.

PLATAFORMA DE NOTÍCIAS

Plataforma de Notícias é um projecto web desenvolvido usando a framework Laravel. Na plataforma os usuários têm opções de visualização, cadastro, edição e remoção de notícias. As notícias podem ou não estar visivéis.

SISTEMA DE GESTÃO INTEGRADA

Sistema de Gestão Integrada é projecto desktop desenvolvido em C# com banco de dados Sql Server. Na sua versão demo, na área administrativa do sistema o administrador tem a possibilidade de fazer a gestão de usuários. Criar novos usuários, definir perfís de acesso, editar dados da empresa, visualizar registos de operações, dentre outras opções.

AGENDA TELEFÓNICA

Agenda Telefónica é um projecto desktop desenvolvido usando a linguagem C# com banco de dados Sql Server. Com design moderno, aplicação é um solução para fazer registos de contactos, assim como anotações.
